# Consultora Management System

Plataforma de ingeniería automatizada para la gestión de ventas, clientes y logística. Diseñada bajo estándares de alta disponibilidad y arquitectura de capas modular.

## 🚀 Arquitectura Core
El sistema opera sobre una topología de **4 capas** (Directivas, Orquestación, Ejecución y Observabilidad), garantizando una separación clara de responsabilidades y un control absoluto del flujo de datos.

## 📱 Frontend SPA & UX
- **Arquitectura**: Single Page Application (SPA) optimizada para dispositivos móviles.
- **Diseño**: Framework visual fundamentado en el estándar **ADN Chapitas**.
- **Estilos**: Tailwind CSS y variables globales para consistencia estética.

## 🛠️ Tecnologías Primarias
- **Backend**: Python (Orquestación / Telemetría), PHP (Servicios Legacy/Soporte).
- **Frontend**: Vanila JS, HTML5 Semantic, Tailwind.
- **Base de Datos**: MySQL / MariaDB (InnoDB).

## 📁 Estructura del Proyecto
- `/admin`: Panel de control administrativo.
- `/dashboard`: Nodo de observabilidad y telemetría.
- `/directives`: Especificaciones técnicas inmutables.
- `/execution`: Lógica operativa de servicios.
- `/utils`: Herramientas de soporte y logging.
