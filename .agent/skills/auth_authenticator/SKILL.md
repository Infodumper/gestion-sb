# Skill: auth_authenticator
## Descripción
Especialista en lógica de verificación de identidades y seguridad de acceso.

## Áreas de Responsabilidad
- Procesamiento de credenciales de usuario (Email/Password).
- Generación y validación de sesiones seguras.
- Manejo de hashes y seguridad de contraseñas.
- Respuesta estructurada AJAX para el formulario de login.
