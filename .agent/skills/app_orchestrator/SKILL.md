# Skill: app_orchestrator
## Descripción
Especialista responsable de la gestión del ciclo de vida de los componentes web dinámicos.

## Áreas de Responsabilidad
- Gestión de modales (`modalApp`, `modalFicha`, etc.).
- Control de carga y descarga de iframes internos.
- Manejo de estados de carga (`spinners`, `loaded` states).
- Orquestación de peticiones AJAX transversales entre aplicaciones.
