# Skill: login_manager

## Descripción
Construye y gestiona todo el flujo de ingreso al sistema, protección de contraseñas, ruteos al Dashboard principal y niveles de acceso (roles), priorizando una experiencia sin fricciones en dispositivos móviles.

## Cuándo usar
Cuando sea necesario implementar un Login, recuperación de contraseñas, creación de usuarios o restricción de permisos.

## Trigger
- "login"
- "iniciar sesión"
- "autenticación"
- "perfiles de usuario"

## Entrada
Credenciales del usuario (email, contraseña) o intento de acceso a rutas protegidas.

## Salida
Sesión válida (`$_SESSION`) o rechazo seguro.
