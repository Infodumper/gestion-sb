# Skill: catalog_manager
## Descripción
Especialista encargado de la administración profunda de productos, categorías y stock.

## Áreas de Responsabilidad
- Gestión transaccional de productos (CRUD).
- Control de inventario y alertas comerciales de stock bajo.
- Agrupación por categorías y catálogos estacionales.
- Manejo de precios y ofertas temporales.
