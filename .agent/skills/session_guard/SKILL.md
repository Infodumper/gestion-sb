# Skill: session_guard
## Descripción
Especialista encargado de proteger la sesión de usuario y la integridad de las rutas del servidor.

## Áreas de Responsabilidad
- Verificación persistente del estado de la sesión (`includes/security.php`).
- Manejo de redirecciones ante accesos no autorizados.
- Control de expiración y logout automático por inactividad.
- Desalojo seguro de variables de sesión.
