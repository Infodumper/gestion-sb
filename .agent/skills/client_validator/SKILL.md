# Skill: client_validator
## Descripción
Especialista encargado de asegurar la integridad y veracidad de la base de datos de clientes.

## Áreas de Responsabilidad
- Validación de números telefónicos para WhatsApp (formato internacional).
- Detección de identidades duplicadas (DNI/Teléfono).
- Normalización de nombres y apellidos (Capitalización adecuada).
- Validación de fechas de cumpleaños (Formato y lógica de edad).
