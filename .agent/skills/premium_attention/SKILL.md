# Skill: premium_attention

## Descripción
Enfocada en la relación post-venta, fidelización y comunicaciones personalizadas para clientes recurrentes.

## Cuándo usar
Para automatizar tareas de marketing relacional, detección de cumpleaños o envíos masivos por WhatsApp.

## Trigger
- "cumpleaños"
- "atención al cliente"
- "enviar WhatsApp"
- "fidelización"
- "clientes VIP"

## Entrada
Base de datos de clientes con fechas de nacimiento y contactos.

## Salida
Listas de difusión personalizadas y enlaces directos de WhatsApp configurados.

## Módulo asociado
`clientes` / `atención` (ubicado en `admin/apps/clientes/atencion_cliente.php`)
